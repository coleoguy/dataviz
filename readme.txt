example.R: This file contains a simple simulated toy example of the seqdef function. It depends on the function.R file as well as the seqdef.csv file

app.R: This contains the Shiny Web App for visualizing the data collected in this project.

seqdef.csv: This is a to dataset that contains dat for the example file

arachnids.csv: This is a subset of the delivered data containing 32k rows of data from arachnids

function.R: This file contains the seqdef function.

Files in this directory require several R packages to run: shiny, viridis, ape, phytools. Once these are installed these files can be run in a sandbox environment.
